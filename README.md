<h2>📌 Overview</h2>
<p>
GenNotesAI is an AI-powered web application that helps students generate high-quality, structured notes instantly. 
It simplifies the learning process by converting raw input into concise, exam-ready content using modern AI technologies.
</p>

<h2>✨ Features</h2>
<ul>
  <li>⚡ Generate notes instantly using AI</li>
  <li>📚 Clean and structured output for better readability</li>
  <li>🔐 Secure authentication system</li>
  <li>🧠 Smart summarization for exam preparation</li>
  <li>🌐 Fully responsive modern UI</li>
</ul>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> React.js, Tailwind CSS</li>
  <li><b>Backend:</b> Node.js, Express.js</li>
  <li><b>Database:</b> MongoDB</li>
  <li><b>Authentication:</b> JWT</li>
  <li><b>AI Integration:</b> OpenAI API</li>
</ul>

<h2>🏗️ Architecture Diagram</h2>

<p align="center">
  <img src="https://raw.githubusercontent.com/dhruvsharmads0506/GenNotesAI/main/assets/architecture.png" 
       alt="Architecture Diagram" width="85%">
</p>

<p align="center">
  <i>High-level architecture showing interaction between frontend, backend, AI engine, and database</i>
</p>

<ul>
  <li>🌐 <b>Frontend (React)</b> handles user input and UI rendering</li>
  <li>⚙️ <b>Backend (Node.js + Express)</b> processes requests and handles business logic</li>
  <li>🧠 <b>AI Engine (OpenAI API)</b> generates structured notes</li>
  <li>🗄️ <b>Database (MongoDB)</b> stores user data and notes</li>
  <li>🔐 <b>Authentication (JWT)</b> secures user sessions</li>
</ul>

<h2>🔄 Application Flow</h2>

<p align="center">
  <img src="https://raw.githubusercontent.com/dhruvsharmads0506/GenNotesAI/main/assets/flow.png" 
       alt="Application Flow" width="85%">
</p>

<p align="center">
  <i>Step-by-step flow of how user input is converted into AI-generated notes</i>
</p>

<ol>
  <li>👤 User logs in / signs up</li>
  <li>📝 User enters topic or content</li>
  <li>📡 Frontend sends request to backend API</li>
  <li>⚙️ Backend validates request & user authentication</li>
  <li>🧠 Backend calls OpenAI API for note generation</li>
  <li>📄 AI processes and returns structured notes</li>
  <li>💾 Notes are optionally stored in MongoDB</li>
  <li>📲 Frontend displays formatted notes to user</li>
</ol>

<h2>🚀 Getting Started</h2>

<h3>1️⃣ Clone the Repository</h3>
<pre>
git clone https://github.com/dhruvsharmads0506/GenNotesAI.git
cd GenNotesAI
</pre>

<h3>2️⃣ Install Dependencies</h3>
<pre>
npm install
</pre>

<h3>3️⃣ Setup Environment Variables</h3>
<pre>
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_api_key
</pre>

<h3>4️⃣ Run the App</h3>
<pre>
npm run dev
</pre>
